测试网易云跟帖

发表于 2017-05-05 | 分类于 生活 | | 阅读次数

写在前面

最近生活琐事太多，加之又换了电脑，一晃半年没有打理了，难得今天抽出空来把整个博客迁移了过来。
这个人真的缺乏执行力。

以及

工作这些年，心态总是在不断的变化
从一开始的浮躁，到渐渐静下心品味生活，再到现在的焦躁不安，真可谓是大起大落。
焦躁的源头大概是因为最近又开始怀疑自己是不是在浪费程序员这宝贵又短暂的生命周期。
拿着拉低行业水平的工资，
尽管热爱前端，却一直无法找到合适的岗位，自学成效低下，还有脖子越来越疼……
人生不如意者十之八九，你是选择随遇而安，还是选择去投身风暴？

正文

这篇没有营养的帖子，主要是为了测试网易云跟帖
其实在网易云跟帖之前我试过多说(2017/6/1停运)，友言(不支持https)，Disqus(被防火长城狙击)
这两年网易给的感觉很奇妙，从阴阳师到网易云音乐，再到网易云跟帖
不管是营销还是产品都做得特别精良，一针见血抓住了市场的核心竞争点
它不再是当初那个靠代理魔兽世界咸鱼翻身的国产网游大厂
是不是丁磊不再沉迷他的养猪场
然而史玉柱更应该回去卖他的脑白金
